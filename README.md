# BuddyShelf-Skill

A skill guide that teaches Claude how to work naturally with BuddyShelf users.

## What is this?

BuddyShelf-Skill is a prompt guide designed to help Claude understand how to interact with BuddyShelf users in a friendly and helpful way. It teaches Claude when to suggest saving work, how to be supportive without being pushy, and how to make the BuddyShelf experience feel like working with a helpful friend. Bascialy it’s helping new coders or vibe coders use versioning in a friendly way.

## Getting Started

1. Check out the `buddyshelf-skill.md` file to see the full skill guide
2. Use this guide as reference material when working with BuddyShelf or training Claude for BuddyShelf interactions
3. The guide covers natural conversation patterns and when to offer helpful reminders about saving work

This project helps make BuddyShelf interactions feel personal and supportive rather than robotic or intrusive.
